# Sound Pad - 音声ファイルランチャー

RustとTauriで作成された音声ファイルランチャーアプリケーションです。SEやBGMの確認・整理に最適です。

## 機能

### 1. 音声ファイルの閲覧と再生
- フォルダを選択すると、対応する音声ファイル（MP3、WAV、OGG、FLAC、M4A、AAC）を自動的に読み込みます
- グリッド表示で見やすいレイアウト
- ワンクリックで音声を再生
- 別の音声をクリックすると、前の音声を自動停止して新しい音声を再生

### 2. フォルダ履歴とブックマーク
- **履歴機能**: 過去に開いたフォルダを自動的に記録（最大10件）
- **ブックマーク機能**: よく使うフォルダをブックマークに登録
- ワンクリックで履歴やブックマークからフォルダを開く
- ブックマークは削除ボタンで簡単に管理可能
- LocalStorageを使用してデータを永続化

### 3. ファイル名の編集
- 各ファイル名を個別に編集可能（クリックして直接入力）
- 複数のファイルを選択して、一括で接頭辞・接尾辞を追加

### 4. ファイルの選択とコピー
- チェックボックスで複数のファイルを選択
- 選択したファイルを指定フォルダにコピー
- コピー前にファイル名の編集が可能
- **コピー完了モーダル**: コピー完了時にモーダルダイアログを表示
  - OKボタンでモーダルを閉じる
  - 「コピー先を開く」ボタンでコピー先フォルダを直接開く

## 使い方

### 開発環境での実行

```bash
npm run tauri dev
```

### プロダクションビルド

```bash
npm run tauri build
```

ビルド後、`src-tauri/target/release/`にインストーラーまたは実行ファイルが生成されます。

## 基本的な操作

1. **フォルダを選択**: 画面上部の「📁 フォルダを選択」ボタンをクリックして、音声ファイルが含まれるフォルダを選択

2. **ブックマークに追加**: フォルダを開いた後、「⭐ ブックマークに追加」ボタンでブックマークに登録

3. **履歴・ブックマークから開く**: 画面上部の履歴またはブックマークエリアから、過去に開いたフォルダやお気に入りのフォルダをクリックして素早く開く

4. **音声を再生**: 各ファイルの「▶ 再生」ボタンをクリックして音声を再生

5. **ファイル名を編集**: ファイル名をクリックして直接編集可能。Enterキーで確定

6. **複数ファイルの一括編集**:
   - チェックボックスで複数のファイルを選択
   - 「✏️ 選択項目の名前を編集」ボタンをクリック
   - 接頭辞と接尾辞を入力

7. **ファイルをコピー**:
   - チェックボックスで必要なファイルを選択
   - 「📋 選択項目をコピー」ボタンをクリック
   - コピー先のフォルダを選択
   - モーダルダイアログで「コピー先を開く」を選択すると、コピー先をアプリで開く

## 技術スタック

- **フロントエンド**: Vanilla JavaScript, HTML, CSS
- **バックエンド**: Rust
- **フレームワーク**: Tauri 2.x
- **音声再生**: rodio
- **ファイル操作**: std::fs, walkdir
- **データ永続化**: LocalStorage API

## 対応音声フォーマット

- MP3
- WAV
- OGG
- FLAC
- M4A
- AAC
